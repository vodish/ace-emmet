Ace Editor - Тест невидимых символов (\u200B и \u200C)

Инструкция:

  1. Нажмите кнопки для вставки невидимых символов в текст
  2. Попробуйте перемещать курсор стрелками влево/вправо
  3. Курсор должен "перескакивать" через невидимые символы
  4. Нажмите "Показать позицию курсора" чтобы увидеть текущую позицию

Текст для теста: Введите: A\u200B\u200CB\u200BC и попробуйте перемещать курсор
